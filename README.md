# File-storage-API

Uma API simples para upload, armazenamento e download de arquivos.  
Permite que clientes enviem arquivos, armazenem de forma persistente e recuperem quando necessário.

## Funcionalidades principais

- Upload de arquivos  
- Download de arquivos
- Armazenamento persistente de arquivos  
- Verificação de usuario com encriptador/desencriptador de senha.

## Tecnologias utilizadas

- Java  
- Maven (gerenciamento de dependências & build)  
- Spring Boot Framework
- Spring Security
- BCryptPasswordEncoder
- PostgreSQL
